# 组件

## Zarm UI [UI组件库]

## 通用组件
- advs [广告组件 支持轮播]
  - 根据广告位标识从服务器端获取广告数据
  - 缓存支持【暂未实现】
- imgLink [图片链接]
- navs [导航栏 支持轮播]
  - 根据导航栏位置标识从服务器端获取导航数据
  - 缓存支持【暂未实现】 
- nullData [无数据提醒]
- userAvatar [用户头像]
- docLink [用户协议链接]
